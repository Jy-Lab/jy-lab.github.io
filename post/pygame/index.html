一. 例行前言



1,电脑安装Python3🐍


2,选择安装一个优秀的IDE代码编辑器👉Pycharm🥰当之无愧吧?一个优秀的编辑器可以让你事倍功半.当然你也可以选择在桌面新建笔记本.txt(摊手)



二. 打开Pycharm添加pygame模块

(Pygame是跨平台Python模块，专为电子游戏设计。包含图像、声音。)


添加步骤如下

左上角 file下的settings...

点击右手边的+号


搜索pygame然后点击install package即可😄






三. import pygame

time呢,拿来控制游戏画面的快慢(帧),
os呢,就是帮我们读取系统文件(例如把主角换成想要的图片).
random则用来产生想要的数字.(例如用于产生1到100一共100个数字)



pycharm里右键run一下,因为我们只定义了基础框架,如果没意外的话,

就会弹出一个黑漆漆的窗口.那么就代表你成功了.😊





四. 不要白费你的功夫

这份代码可以保存为一个副本,要写pygame小游戏的时候可以快速copy一下.

从中添加,改造成一切你想要的模样.😉
