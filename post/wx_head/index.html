微信官方修头像究竟在背后干了些什么?!🤔


一·读取头像

#没有PIL模块的>>>pip install PIL 😎使用Pycharm的搜Pillow即可


二·读取本地准备好的想加上去角落的图片 （例如国旗？）


三· 一步到位，贴到头像上去


分解一下:

img.paste(img,(box),mask=mask)这个用法的参数第一个img是图片，box是放在哪个像素位置，mask就是掩盖，面具的意思.我是已经PS好的透明底的国旗,所以mask就是图片本身(这个很关键)


以头像整体为坐标，把Flag resiza（缩放）成头像的四分之一.Image.ANTIALIAS是磨边.


于是有
(Flag.resize((FlagW,FlagH), Image.ANTIALIAS)

至于放在box的位置，画个图就一目了然
细心的其实可以发现,微信头像都是正方形的


((int(Weight*3/4)),(int(Height*3/4)))选择3/4处.顺手转成int整数类型.

paste贴好之后,保存.可以收工.😁
